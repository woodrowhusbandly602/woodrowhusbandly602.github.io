This repository provides cursor sets inspired by the game Dota 2. You can replace your standard Windows mouse pointer with icons shaped like heroes, items, or interface designs from the game. These files work on any standard Windows installation.

## 📁 What is inside

The cursor packs contain graphical files that represent your mouse pointer. When you move your mouse, the files show the Dota 2 theme instead of the default Windows arrow. Each pack includes different pointer states, such as:

* The standard pointer (the arrow)
* The link selector (the hand)
* The text selector (the beam)
* The busy icon (the loading circle)

## 🛠️ Installing on Windows

Windows stores cursors in a specific folder. You need to extract the files you downloaded and move them into this storage area. 

### Extract the files
1. Find the .zip file you downloaded.
2. Right-click the file.
3. Select "Extract All."
4. Choose a destination folder and click "Extract."
5. Open the new folder to see the cursor files.

### Add cursors to Windows
1. Open the File Explorer.
2. Go to your C: drive folder.
3. Open the "Windows" folder, then the "Cursors" folder.
4. Copy the cursor files from your extracted folder and paste them into this C:\Windows\Cursors folder.
5. Windows might ask for your permission to move these files. Click "Continue" to allow the move.

## ⚙️ Applying the new theme

Once the files live in your system folder, you can change your pointer settings. 

1. Click the Start button.
2. Type "Mouse settings" and press Enter.
3. Click on "Additional mouse options" on the right side of the window.
4. A new window named "Mouse Properties" appears.
5. Click the "Pointers" tab at the top.
6. Click the "Browse" button.
7. Locate the folder where you pasted the Dota 2 cursors.
8. Select the cursor you want to use for the "Normal Select" function.
9. Click "Open."
10. Repeat these steps for other pointer types like "Link Select" or "Text Select" to maintain the theme throughout your system.
11. Click "Apply" and then "OK" to save your changes.

## 🖥️ System requirements

These cursors work on most modern versions of Windows. You need:
* A version of Windows 10 or Windows 11.
* Basic access to system folders.
* A few megabytes of free space on your hard drive. 
* Mouse settings access through your Control Panel.

## ℹ️ Troubleshooting

If you do not see the changes, restart your computer. Sometimes Windows needs a refresh to show new cursor icons. 

If a cursor looks too large or too small, you can adjust your scaling settings in Windows. Right-click your desktop, select "Display settings," and check the "Scale and layout" percentage. Cursors look sharpest at 100% or 125% scale.

If you want to revert to the default Windows cursors, return to the "Pointers" tab in the "Mouse Properties" menu. Click "Use Default" and your system will switch back to the standard cursor icons immediately.
