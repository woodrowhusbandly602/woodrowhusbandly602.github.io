The details

Drop the kunai, pin the target. Kunai Drop is a physics puzzler where every throw ricochets — walls, bells, bamboo, and the occasional unlucky lantern all change the blade's path.

Two hundred puzzles that start gentle and end devious. Solutions are never twitchy; they are thought through.

  • 200 physics puzzles
  • Deterministic physics — plan, don't pray
  • Hint scrolls when you're stuck
  • Plays fully offline
